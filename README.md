# Лабораторна робота №3: Java & XML

**Виконав:** студент гр. КБ-22 Бежан Гліб

## Опис проєкту
Проєкт містить вирішення **Завдання №0** з лабораторної роботи. 
Мета програми: обробка датасету популярних імен дітей у місті Нью-Йорк (`names_data.xml`).

## Виконані кроки:
1. Здійснено парсинг датасету за допомогою **SAX-парсера** (клас `NamesSAXHandler`). Програма автоматично збирає унікальні теги та всі представлені національні групи.
2. Проведено фільтрацію даних для обраної етнічної групи та сортування топ-імен за рейтингом.
3. Згенеровано новий XML-документ (`names_result.xml`) за допомогою **DOM-парсера**, який містить відфільтровані результати.
4. Створено XSD-схему (`names_schema.xsd`) для перевірки структури вхідного документа.
